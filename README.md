这是一款基于 **Python** 和 **Pygame** 开发的 2048 游戏 AI 引擎。
**声明**：利用 AI (Gemini)  生成了 GUI 框架代码，核心 AI 算法逻辑由本人独立实现。

## 🚀 如何在本地运行

### 1. 环境准备
确保你的电脑已安装 **Python 3.8** 或更高版本。

### 2. 克隆项目
打开终端（或 CMD），运行以下命令下载本项目：
```bash
git clone https://github.com/你的用户名/你的仓库名.git
cd 你的仓库名
```

### 3. 安装依赖
本项目主要依赖 `pygame` 库。你可以通过以下命令快速安装：
```bash
pip install -r requirements.txt
```

### 4. 启动程序
运行 `main.py` 即可看到 AI 开始自动进行游戏：
```bash
python main.py
```
### 5. Windows 用户快捷启动
为了方便操作，你可以直接双击根目录下的 `start.bat` 文件（如果已创建），它会自动检测依赖并启动游戏。

## 📂 文件结构说明
项目由以下核心文件组成：

**`main.py`**: 程序入口。初始化游戏循环，连接 AI 决策与 UI 渲染。

**`ai.py`**: AI 核心。

**`logic.py`**: 提供一行的移动结果和评分、移动后状态等程序逻辑。

**`gui.py`**: 图形界面。负责窗口绘制、方块动画及文字显示。

**`constants.py`**: 配置文件。定义程序的各种参数。

***tips***: 请谨慎修改搜索深度`DEPTH`。对于超过5的搜索深度，您的计算机可能卡死。
